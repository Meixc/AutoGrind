Download the Package: Click on the AutoGrind_V1.0.zip file under the "Assets" section of the [Latest Release Page]([Paste Your Release URL Here]).

Extract the Files: Unzip the downloaded folder. It will contain:

Start.bat (The launcher)

AutoGrind.txt (The script content)

Run the Launcher: Double-click on Start.bat.

Follow On-Screen Instructions: A black window will appear, displaying the message "Your Script is Running" and an instruction to copy the command from the newly opened text file and paste it into their Roblox Executor.
